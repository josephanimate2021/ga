	<h3>Download Instructions</h3>
	<h4>Windows:</h4>
	<ol>
		<li>Click on button with screen size <script type="text/javascript">document.write("(your current screen size is <em>"+screen.width+" x "+screen.height+"</em>)");</script>.</li>
		<li>Open the File Browser application and then Drag the picture inside the application and then double click on it. Once in, Right click on the picture, and move your move to <em>"Set As >"</em> and then click on  <em>"Background"</em>.</li>
	</ol>
	<h4>Mac (OSX):</h4>
	<ol>
		<li>Hold down the control key and click on the button with your screen size <script type="text/javascript">document.write("(your current screen size is <em>"+screen.width+" x "+screen.height+"</em>)");</script>.</li>
		<li>Select <em>"Download Linked File As..."</em> from the menu.</li>
		<li>Select <em>"Desktop"</em> from the <em>"Where"</em> menu and press save.</li>
		<li>Hold down the control key and click on your existing desktop.</li>
		<li>Select <em>"Change Desktop Background..."</em> from the menu.</li>
		<li>Select <em>"Desktop"</em> and select the file which you have saved.</li>
	</ol>
